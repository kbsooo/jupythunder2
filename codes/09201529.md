### 15:30:10 · 에이전트

Insert sort는 정렬 알고리즘 중 하나로, 배열의 요소를 하나씩 비교하여 정렬합니다. 시간 복잡도는 O(n^2)입니다.

#### 계획

- 요소 하나씩 비교합니다.
- 작은 요소를 앞으로 이동합니다.
- 정렬된 배열을 반환합니다.

#### 코드 셀 cell-1 · Insert sort 함수 정의

```python
def insert_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr
```
